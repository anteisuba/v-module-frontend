## P0: 直ちに処理する項目

- `/blog` と `/shop` の固定 slug リダイレクトを撤去し、共通入口方針を明確化
- `PageConfig.links` と実際のレンダリング挙動を一致させる
- 最小限の自動テスト基盤を追加する

## P1: 機能の閉ループ

- 注文詳細ページ
- 注文検索、フィルタ、エクスポート
- 注文作成 / 状態更新メール通知
- 決済ゲートウェイ統合
- コメント審査画面
- `MediaAsset` を使ったメディアライブラリ UI

## P2: 体験拡張

- カートと複数商品決済
- 在庫アラート
- 配信スケジュール
- SEO と公開入口の再整理
- 監視、ログ、エラートラッキング強化
