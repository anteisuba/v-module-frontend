## 実装済み

- ユーザー機能: 登録、ログイン、ログアウト、パスワードリセット、管理画面ガード
- ページ設定: 下書きと公開設定の分離、公開ページは `publishedConfig` を参照
- 管理画面編集骨格: CMS、ブログ、ショップの一覧 / 詳細編集をタブ + 単一展開の折りたたみパネルに統一
- 公開ページ: テーマカラー、フォント、背景、Hero、News、Video などの設定駆動表示
- ニュース: 一覧、詳細、管理画面編集と公開 API
- ブログ: 管理画面一覧 / 編集、公開一覧 / 詳細、いいね API、コメント API と UI
- ショップ: 商品管理、ショップ一覧 / 商品詳細背景の編集、公開商品一覧 / 詳細、公開チェックアウト API `POST /api/shop/checkout`、売り手向け注文一覧、注文状態更新
- メディアアップロード: 管理画面アップロード、ローカル / R2 分岐、`MediaAsset` 記録
- i18n 基盤: `zh`, `ja`, `en`

## 部分実装 / ギャップあり

- 注文成功ページはプレースホルダーで、注文詳細を取得していない
- `/blog`, `/shop` は現在 `/u/xiuruisu/...` へ固定リダイレクト
- `PageConfig` に `links` type は残っているが、レンダラは `null` を返す
- `MediaAsset` は保存されるが、媒体ライブラリ UI はない
